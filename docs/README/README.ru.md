# CloseUp

[English](../../README.md) · [简体中文](README.zh-CN.md) · [繁體中文](README.zh-TW.md) · [日本語](README.ja.md) · [Français](README.fr.md) · [Deutsch](README.de.md) · [Español](README.es.md) · [Português](README.pt-BR.md) · **Русский**

**Верните элементы управления в Mission Control.** CloseUp накладывает элементы управления окнами на нативный Mission Control в macOS — закрывайте, сворачивайте, масштабируйте, скрывайте или завершайте любое окно, не покидая обзор, — и добавляет полное управление с клавиатуры. Бесплатно, нативно и с открытым исходным кодом.

## Возможности

- **Закрытие из обзора** — наведите указатель на миниатюру окна в Mission Control и нажмите красный ×, чтобы мгновенно закрыть окно, не переключаясь на него.
- **Все действия с окном** — сворачивайте, масштабируйте, скрывайте программу или завершайте её работу; каждое действие — это дополнительная кнопка, которую можно включить или отключить.
- **Управление с клавиатуры** — выполняйте над окном под указателем нативные действия: ⌘W закрыть, ⌘M свернуть, ⌘F масштабировать, ⌘H скрыть, ⌘Q завершить — все они переназначаемы.
- **Пакетные действия** — ⌥⌘W закрыть все, ⌥⌘M свернуть все, ⌥⌘H скрыть все, кроме окна под указателем.
- **Девять языков** — English, 简体中文, 繁體中文, 日本語, Français, Deutsch, Español, Português, Русский, переключаются прямо в программе и применяются на лету.
- **Нативный и ненавязчивый** — пассивный оверлей, который никогда не перехватывает собственную обработку клавиш Mission Control; работает только в строке меню, без значка в Dock.
- **Автоматическое обновление** — подписанные и нотаризованные выпуски через Sparkle.

## Требования

- macOS 14.0 или новее
- Apple Silicon или Intel (универсальный двоичный файл)
- Разрешение «Универсальный доступ» (CloseUp считывает окна и действует с ними через API Универсального доступа; запись экрана при этом не ведётся)

## Установка

Загрузите последний файл `.dmg` из раздела [Releases](https://github.com/oomol-lab/CloseUp/releases), откройте его и перетащите CloseUp в папку «Программы». При первом запуске предоставьте доступ к Универсальному доступу в «Системных настройках» → «Конфиденциальность и безопасность» → «Универсальный доступ» — CloseUp откроет нужную панель за вас.

## Использование

Откройте Mission Control обычным способом (смахните вверх тремя или четырьмя пальцами либо нажмите клавишу Mission Control). Наведите указатель на любое окно, чтобы отобразить кластер элементов управления, или используйте клавиатуру:

| Действие | Сочетание клавиш | Применяется к |
|---|---|---|
| Закрыть окно | ⌘W | окно под указателем |
| Свернуть окно | ⌘M | окно под указателем |
| Масштабировать окно | ⌘F | окно под указателем |
| Скрыть программу | ⌘H | окно под указателем |
| Завершить программу | ⌘Q | окно под указателем |
| Закрыть все окна | ⌥⌘W | все окна |
| Свернуть все окна | ⌥⌘M | все окна |
| Скрыть все, кроме этого | ⌥⌘H | все программы, кроме той, что под указателем |

Каждое сочетание клавиш можно переназначить в разделе «Настройки» → «Сочетания клавиш», а глобальная горячая клавиша позволяет включать и отключать CloseUp из любого места.

## Настройки

- **Основные** — включение и отключение, запуск при входе в систему, отображаемые кнопки управления и язык программы.
- **Сочетания клавиш** — переназначение каждого действия и глобального переключателя.
- **Разрешение** — состояние Универсального доступа и предоставление доступа в одно нажатие.
- **Обновления** — автоматическая проверка и кнопка «Проверить наличие обновлений» вручную.

## Сборка из исходного кода

```bash
brew install xcodegen
make build      # Отладочная сборка (отдельная идентичность «CloseUp Dev»)
make test       # модульные тесты + проверки локализации
make run        # сборка и запуск
make dmg        # упаковка в .dmg для установки перетаскиванием
```

Проект Xcode генерируется из `project.yml` с помощью XcodeGen и не добавлен в репозиторий. См. [docs/DESIGN.md](docs/DESIGN.md) для описания архитектуры и [docs/RUNBOOK.md](docs/RUNBOOK.md) для описания процесса выпуска.
